# Python教育用サンプル

## 初めに

### 動作環境

* Python 3.10.x
* Pythonパッケージ管理 Poetry

### セットアップ

Python はインストール済みの前提です。

まずはPythonのバージョンを確認します。

```shell
python -V
```

`Python 3.10.x`と表示され、バージョンが3.10系であること確認して下さい。

以下のコマンドで`Poetry`をインストールします。

```shell
pip install poetry
poetry install
```
